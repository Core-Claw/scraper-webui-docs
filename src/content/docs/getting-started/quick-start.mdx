Get your first Worker running in just 5 minutes!

## Prerequisites

- A CoreClaw account ([Sign up here](https://console.coreclaw.com/sign-up))
- Basic understanding of web scraping concepts

## Step-by-Step Guide

### 1. Create an Account

Go to [CoreClaw](https://coreclaw.com) and click "Sign Up". Fill in your details and verify your email.

### 2. Find a Worker

Navigate to the Worker Store to browse or search for the website you want to scrape. Click on a Worker to view details, including:

- Description and use cases
- Input parameters
- Sample output
- Pricing

### 3. Run Your First Worker

Click "Run Worker" and configure the input parameters. Click "Start" and wait for the run to complete.

### 4. Get Your Data

Go to the "Runs" section to find your completed run. Download data in your preferred format:

- JSON
- CSV
- Excel
- Or access via API

## Understanding the Results

After a run completes, you'll have access to:

| Data Type      | Description                             |
| -------------- | --------------------------------------- |
| **Results**    | The extracted data in structured format |
| **Logs**       | Execution logs for debugging            |
| **Statistics** | Performance metrics and resource usage  |
