Get your first scraper running in just 5 minutes!

## Prerequisites

- A CafeScraper account ([Sign up here](https://console.cafescraper.com/sign-up))
- Basic understanding of web scraping concepts

<Steps>
1. **Create an Account**

   Go to [CafeScraper](https://cafescraper.com) and click "Sign Up". Fill in your details and verify your email.

2. **Find a Scraper**

   Navigate to the Scraper Store to browse or search for the website you want to scrape. Click on a scraper to view details, including:
   
   - Description and use cases
   - Input parameters
   - Sample output
   - Pricing

3. **Run Your First Scraper**

   Click "Run Scraper" and configure the input parameters. Click "Start" and wait for the run to complete.

4. **Get Your Data**

   Go to the "Runs" section to find your completed run. Download data in your preferred format:
   
   - JSON
   - CSV
   - Excel
   - Or access via API
</Steps>

## Understanding the Results

After a run completes, you'll have access to:

| Data Type | Description |
|-----------|-------------|
| **Results** | The extracted data in structured format |
| **Logs** | Execution logs for debugging |
| **Statistics** | Performance metrics and resource usage |
