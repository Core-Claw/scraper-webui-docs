Welcome to CafeScraper! This guide will help you get up and running quickly.

## What is CafeScraper?

CafeScraper is a powerful web scraping platform that allows you to:

- **Extract data** from any website with ease
- **Run scrapers** in the cloud without infrastructure management
- **Access data** via API or download in various formats
- **Schedule tasks** for automated data collection

## Who is this for?

| User Type | Description |
|-----------|-------------|
| **Data Analysts** | Extract data for analysis and reporting |
| **Developers** | Build and deploy custom scrapers |
| **Business Users** | Get data without technical knowledge |
| **Researchers** | Collect data for academic research |
