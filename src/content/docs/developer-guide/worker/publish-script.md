### Submission & Usage

Script types:

- **Private Script**: After submission, no review is required. It can be run directly in the executor and is only visible to you.
- **Public Script**: After submission, it requires review. Once approved, it will be publicly listed in the CoreClaw Store and available for all users to search and use.

**File format requirement**: Script execution files must be in ZIP compressed format and comply with platform specifications.

### Publishing a Private Script

After creation, it can be run in the test environment without review, but it is limited to testing purposes and is not publicly visible.

### Publishing to the Store

If your script passes review and is listed on the [CoreClaw Store](https://coreclaw.com/store), you can earn revenue (see the CoreClaw Script Revenue Rules for details).

**Review Process**: After submission, it enters the review queue and is automatically listed upon approval.

**Publishing Tips**:

- Provide a clear script description to help users find it easily.
- Set reasonable default parameters to lower the barrier to entry.
- Prioritize publishing scripts for popular platforms (e.g., e-commerce, social media) to increase visibility.
