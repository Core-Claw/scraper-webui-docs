## Memory and CPU

Higher memory allocations usually provide stronger CPU performance.

**Note**: The following table lists all available device configurations and memory options.

| Memory | CPU   |
| ------ | ----- |
| 512    | 0.125 |
| 1024   | 0.250 |
| 2048   | 0.500 |
| 4096   | 1.000 |
| 8192   | 2.000 |
| 16384  | 4.000 |
