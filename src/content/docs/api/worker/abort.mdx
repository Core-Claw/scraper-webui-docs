**Method:** `POST`

**Endpoint:** `/api/v1/scraper/abort`

Send the request body with **Content-Type: application/json**.

## Try it

<ApiPlayground method="POST" path="/api/v1/scraper/abort" />

## Request Example

```json
{
    "run_slug": "YOUR_RUN_SLUG"
}
```

#### Parameters

| Parameter | Type   | Required | Description           |
| --------- | ------ | -------- | --------------------- |
| run_slug  | String | Yes      | Unique run identifier |

## Response Example

```json
{
    "code": 0,
    "message": "success",
    "data": null
}
```

#### Response Fields

| Parameter | Example | Type    | Description        |
| --------- | ------- | ------- | ------------------ |
| code      | 0       | Integer | Global status code |
| message   | success | String  | Response message   |
| data      | null    | Null    | Empty payload      |
