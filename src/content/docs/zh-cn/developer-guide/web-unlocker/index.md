# 网页解锁器

CoreClaw 的网页解锁器帮助您绑过反爬虫措施，访问原本被屏蔽的内容。

## 功能特性

- **指纹浏览器** - 轮换浏览器指纹以避免被检测
- **代理网络** - 通过全球代理网络访问内容
- **验证码绑过** - 自动处理 CAPTCHA 和验证挑战
- **SOCKS5 网络代理** - 配置自定义代理设置
