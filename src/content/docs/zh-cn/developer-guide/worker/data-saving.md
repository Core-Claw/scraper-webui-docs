### 数据保存方式

在 CoreClaw 平台中，用户执行抓取任务后，系统会根据任务设置自动保存数据结果。平台支持将抓取数据导出为以下常用格式：

- JSON
- Excel

用户可以根据自身需求下载或调用 API 获取数据。

### 数据存储期限

平台默认会对用户抓取的数据进行一定期限的存储，具体存储策略可能根据账户类型或服务套餐有所不同。建议用户在完成数据处理后及时下载并备份所需数据，以避免因存储周期结束导致的数据丢失。

### 数据管理权限

用户对其账户下生成的数据拥有管理权限，可以：

- 查看已执行任务的数据结果
- 下载或删除历史数据
- 通过控制台管理抓取内容

平台不会未经用户授权擅自使用用户数据。

### 数据安全与保护

CoreClaw 重视用户数据安全，采取技术措施保障数据存储与传输过程的安全性。平台会按照相关安全规范处理数据，但用户仍需自行负责对敏感数据的管理与备份。

### 注意事项

- 建议定期下载重要数据进行本地备份
- 数据保存策略可能因服务版本或功能更新而调整
- 若账户长期未使用，系统可能会根据存储政策清理历史数据
