### 功能概述

用户登录控制台后，进入【Script】模块，可以查看并管理个人已创建的脚本列表。

### 操作步骤

1. **进入脚本详情页**：点击任意一个脚本名称。
2. **选择设置与发布**：在脚本详情页中，找到并点击【**设置与发布**】选项卡。
3. **执行更新操作**：在发布页面中，您可以进行以下操作：
   - 重新上传脚本的 **ZIP 文件**
   - 更新脚本的功能描述及相关信息
4. **提交审核**：确认信息无误后，点击"发布"按钮提交脚本更新。
5. **平台审核**：提交后的脚本将进入平台审核流程。
6. **发布完成**：审核通过后，新版本的脚本将对其他用户可见并可用。
