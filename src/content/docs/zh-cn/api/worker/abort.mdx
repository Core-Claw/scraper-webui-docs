<ApiPlayground method="POST" path="/api/v1/scraper/abort" />


## 请求示例

```json
{
    "run_slug": "YOUR_RUN_SLUG"
}
```

#### 参数说明

| 参数     | 类型   | 必填 | 说明           |
| -------- | ------ | ---- | -------------- |
| run_slug | String | 是   | 运行唯一标识符 |

## 响应示例

```json
{
    "code": 0,
    "message": "success",
    "data": null
}
```

#### 响应字段

| 参数    | 示例    | 类型    | 说明       |
| ------- | ------- | ------- | ---------- |
| code    | 0       | Integer | 全局状态码 |
| message | success | String  | 响应消息   |
| data    | null    | Null    | 空数据     |
