**方法：** `POST`

**端点：** `/api/v1/run/last/log`

使用 **Content-Type: application/json** 发送请求体。

## 在线试用

<ApiPlayground method="POST" path="/api/v1/run/last/log" />


## 请求示例

```json
{
    "run_slug": "YOUR_RUN_SLUG"
}
```

#### 参数说明

| 参数     | 示例值        | 类型   | 必填 | 说明           |
| -------- | ------------- | ------ | ---- | -------------- |
| run_slug | YOUR_RUN_SLUG | String | 是   | 运行唯一标识符 |

## 响应示例

```json
{
    "code": 0,
    "message": "success",
    "data": {
        "all_logs_url": "https://smpfile.coreclaw.com/log/all_log_7cb43d81-38d5-47b6-a313-4e1423d4c28a_1773383790454",
        "list": [
            {
                "type": 2,
                "group": "28596067108978688",
                "content": "SYSTEM: 子任务-[28596067108978688] - 正在准备执行环境。",
                "timestamp": 1773383784438
            }
        ],
        "result_count": 4
    }
}
```

#### 响应字段

| 参数         | 示例    | 类型    | 说明 |
| ------------ | ------- | ------- | ---- |
| code         | 0       | Integer | 全局状态码 |
| message      | success | String  | 响应消息 |
| data         | -       | Object  | 响应数据 |
| all_logs_url | -       | String  | 完整日志文件 URL |
| list         | -       | Array   | 摘要日志条目 |
| result_count | 4       | Integer | 采集总行数 |

##### `list` 字段

| 参数      | 示例 | 类型    | 说明 |
| --------- | ---- | ------- | ---- |
| type      | 1    | Integer | 日志级别：1 调试，2 信息，3 警告，4 错误 |
| group     | -    | String  | 日志组标识符 |
| content   | -    | String  | 日志内容 |
| timestamp | -    | Integer | 日志时间戳（毫秒） |
