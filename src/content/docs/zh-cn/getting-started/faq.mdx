## 常规问题

### CafeScraper 是什么？

CafeScraper 是一个基于云端的网页抓取平台，让您无需管理基础设施即可从网站提取数据。我们提供预构建的爬虫和开发者构建自定义爬虫的工具。

### CafeScraper 如何工作？

1. **查找或构建**：选择预构建爬虫或创建自己的爬虫
2. **配置**：设置抓取任务的输入参数
3. **运行**：在我们的云基础设施中执行爬虫
4. **获取数据**：下载结果或通过 API 访问

### 网页抓取合法吗？

网页抓取对于公开数据通常是合法的，但法规因地区而异。请始终：
- 检查网站的服务条款
- 尊重 robots.txt
- 未经同意不要抓取个人数据
- 使用速率限制避免服务器过载

## 账户问题

### 如何创建账户1. 访问 [CafeScraper](https://cafescraper.com)
2. 点击"注册"
3. 输入邮箱并创建密码
4. 验证您的邮箱地址

### 如何重置密码？

在登录页面点击"忘记密码"并按照发送到邮箱的说明操作。

### 可以更改邮箱地址吗？

可以，进入账户设置 > 个人资料更新邮箱地址。您需要验证新邮箱。

## 支付问题

### 定价如何计算？

CafeScraper 采用基于使用量的定价模型：
- **计算单元**：按爬虫运行收费，基于执行时间
- **数据传输**：按 GB 数据传输收费
- **存储**：按 GB/月收费存储数据

详情请查看[定价规则](/zh-cn/user-guide/scraper-pricing-rules/)。

### 接受哪些支付方式？

我们接受：
- 信用卡/借记卡（Visa、Mastercard、American Express）
- PayPal
- 银行转账（企业账户）

### 如何获得退款？

在购买后 7 天内联系我们的支持团队。退款按个案处理。详情请查看我们的[服务条款](/zh-cn/platform-policies/terms-of-service/)。

## 技术问题

### 为什么我的爬虫失败？

常见原因：
- **反爬保护**：启用网页解锁器
- **无效输入**：检查您的参数
- **网站变更**：目标网站可能已更改
- **速率限制**：降低请求频率

查看运行日志获取具体错误信息。

### 如何调试爬虫1. 在仪表板中查看运行日志
2. 使用预览功能查看截图
3. 使用最小输入参数测试
4. 如问题持续，联系支持

### 速率限制是多少？

API 速率限制因套餐而异：
- **免费版**：100 请求/分钟
- **专业版**：500 请求/分钟
- **企业版**：自定义限制
