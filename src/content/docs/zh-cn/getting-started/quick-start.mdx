5 分钟内运行您的第一个爬虫！

## 前提条件

- 一个 CafeScraper 账户（[点击注册](https://console.cafescraper.com/sign-up)）
- 基本的网页抓取概念理解

<Steps>

1. **创建账户**

   访问 [CafeScraper](https://cafescraper.com) 并点击"注册"。填写您的信息并验证邮箱。

2. **查找爬虫**

   进入爬虫商店浏览或搜索您想要抓取的网站。点击爬虫查看详情，包括：
   
   - 描述和使用场景
   - 输入参数
   - 示例输出
   - 定价

3. **运行您的第一个爬虫**

   点击"运行爬虫"并配置输入参数。点击"开始"并等待运行完成。

4. **获取数据**

   进入"运行记录"部分找到您完成的运行。以下载您偏好的格式下载数据：
   
   - JSON
   - CSV
   - Excel
   - 或通过 API 访问

</Steps>

## 理解结果

运行完成后，您将获得：

| 数据类型 | 描述 |
|-----------|-------------|
| **结果** | 结构化格式的提取数据 |
| **日志** | 用于调试的执行日志 |
| **统计** | 性能指标和资源使用情况 |
