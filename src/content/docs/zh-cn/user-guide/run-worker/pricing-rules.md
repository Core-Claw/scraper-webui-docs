## 按用量付费

您只需为 Worker 运行时**实际消耗**的平台资源付费，无额外隐形费用，价格透明。

**只为成功结果付费，失败请求完全免费。**

查看您订阅的计划，根据不同计划可以享受不同的优惠和功能支持。

![订阅计划](@/assets/docs/69.png)

![计划详情](@/assets/docs/70.png)

:::note
图示价格仅供参考，会根据平台政策有所调整，详细定价信息请访问 [coreclaw.com/pricing](https://www.coreclaw.com/pricing) 或 [console.coreclaw.com/wallet](https://console.coreclaw.com/wallet)。
:::

## 如何估算成本？

虽然很难在运行前精准预测资源消耗，但您可以在控制台的运行历史处查看每次运行后的费用消耗记录。

![费用消耗记录](@/assets/docs/71.png)

:::note[最佳实践]
我们建议您进行小规模测试（例如仅抓取少量页面）。通过试运行，您可以直观地评估资源消耗情况，从而推算出大规模运行的预算。
:::
