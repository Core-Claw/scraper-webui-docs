## 按用量付费

您只需为 Worker 运行时**实际消耗**的平台资源付费，无额外隐形费用。

**只为成功结果付费，失败请求完全免费。**

![Worker 定价配置](@/assets/docs/img_4.jpg)

## 如何估算成本？

虽然很难在运行前精准预测资源消耗，但您可以随时查看每次运行后的详细账单。
