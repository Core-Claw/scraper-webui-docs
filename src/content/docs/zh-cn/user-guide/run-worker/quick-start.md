5 分钟内运行您的第一个 Worker！

## 前提条件

- CoreClaw 账户（[在此注册](https://console.coreclaw.com/sign-up)）
- 基础的网络数据采集概念理解

## 分步指南

### 1. 创建账户

前往 [CoreClaw](https://coreclaw.com) 并点击"注册"。填写您的信息并验证邮箱。

### 2. 查找 Worker

进入 Worker Store 浏览或搜索您要采集的网站。点击 Worker 查看详情，包括：

- 描述和使用场景
- 输入参数
- 示例输出
- 定价

![Worker Store 页面](@/assets/docs/img_26.jpg)

### 3. 运行您的第一个 Worker

点击"运行 Worker"并配置输入参数。点击"开始"并等待运行完成。

![Worker 运行详情](@/assets/docs/img_30.jpg)

### 4. 获取数据

进入"运行记录"部分找到已完成的运行。以您偏好的格式下载数据：

- **JSON** - 适合开发者和 API 集成
- **CSV** - 适合电子表格和数据分析

## 理解结果

运行完成后，您可以访问：

| 数据类型 | 说明 |
| -------- | ---- |
| **结果** | 结构化格式的采集数据 |
| **日志** | 用于调试的执行日志 |
| **统计** | 性能指标和资源使用 |
