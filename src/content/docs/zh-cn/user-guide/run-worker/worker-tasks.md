Task 支持您为同一 Worker 创建并保存多个预设配置，方便直接调用和复用。例如，您可以配置一个"商品抓取"任务，用于提取 Amazon 上多个商品的基本信息、评论等内容，只需一次性设定好目标链接和其他参数，该配置即可重复用于抓取不同商品。

您可以直接在 CoreClaw 控制台中管理、运行这些任务配置，并查看其执行记录，实现高效的任务管理与复用。
