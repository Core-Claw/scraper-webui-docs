## 一、下载数据的步骤

##### 第一步：进入任务页面

- 登录您的账户
- 进入「Runs」
- 点击对应的 Worker 标题或者 Run ID

##### 第二步：查看运行结果

在任务详情页中，您可以看到：

- 运行状态
- 抓取条数（Items Count）
- 执行时间
- 日志记录（Logs）

确认状态为成功（Completed）后，即可点击导出下载数据。

##### 选择下载格式

系统通常支持两种结构化数据格式：

- 📊 XLSX（Excel 文件）
- 📦 JSON（适合开发者使用）
- 🗂 API 访问（用于系统对接）

点击格式按钮，选择所需格式即可。

---

## 二、API 方式获取数据（适用于企业及开发者）

如果您需要自动化获取数据，可以使用 API 方式：

API属于定制服务，详情可以咨询平台客服

- 进入 Worker 详情页
- 找到 API Endpoint
- 使用 Token 进行授权访问
- 通过 GET 请求拉取结果数据

适用于：

- 定时数据同步
- 系统对接
- BI 工具连接
- 自动化报表生成

---

## 三、常见问题

##### ❓ 下载按钮不可点击怎么办？

可能原因：

- 任务尚未完成
- 抓取结果为空
- 账户余额不足
- 权限受限

建议刷新页面或检查运行状态，如还是无法下载请联系平台处理。

##### ❓ 数据为空怎么办？

请检查：

- 输入参数是否正确
- 目标网址是否支持抓取
- 是否存在反爬限制

可以查看日志获取详细报错信息。
