---
title: Workder Task
description: Task 支持您为同一 Workder 创建并保存多个预设配置，方便直接调用和复用。
---

Workder tasks allow you to create and save **multiple preset configurations** for the same Workder, making it easy to reuse and run them on demand.

For example, you can configure a **“Product Scraping”** task to extract basic product information, reviews, and other data from Amazon. Once the target URLs and parameters are set, the same configuration can be reused to scrape different products.

You can manage and run these task configurations directly in the **CoreClaw Console**, as well as view their execution history, enabling efficient task management and reuse.
