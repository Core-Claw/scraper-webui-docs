## 1. Worker Discovery Entries

The platform provides **two entries for discovering Workers**, allowing users to choose based on their usage scenario:

1. Official Worker Store
2. Console Worker Store Menu

Both entries display the same Worker information and support browsing and search.

---

## 2. Finding Workers in the Official Worker Store

##### 2.1 Access the Worker Store

Users visit the official platform website and navigate to the **Worker Store** page via the navigation bar or related entries.

![](@/assets/docs/img_26.jpg)

##### 2.2 Browse Worker Categories

On the **Worker Store** page, Workers are typically organized by functionality or application scenarios, such as:

- Website data collection
- Search engines
- E-commerce platforms
- Social media
- General-purpose data collection tools

Users can quickly browse Workers of a specific type by selecting a category.

![](@/assets/docs/img_27.jpg)

##### 2.3 Worker Search

The **Worker Store** supports search through the search box, including:

- Fuzzy search by **Worker name**
- Fuzzy search by **Worker description / keywords**

After entering keywords, the system automatically matches and displays relevant Workers.

---

## 3. Finding Workers via the Console Workers Store Menu

##### 3.1 Access the Worker Store Page

After logging in to the console, click **Worker Store** in the left-side menu to enter the Worker management page.

![](@/assets/docs/img_28.jpg)

##### 3.2 View the Worker List

On the Worker Store page, users can view all available Workers provided by the platform, including:

- Worker name
- Worker summary
- Worker status
- Supported features

##### 3.3 Worker Search

The Worker Store page also supports search functionality, including:

- Fuzzy search by Worker name
- Fuzzy search by **Worker content** / **description keywords**

Search results are updated in real time, helping users quickly locate the desired Worker.

![](@/assets/docs/img_29.jpg)

---

## 4. Viewing Worker Details

Whether in the **Official Worker Store** or the **Console Worker Store page**, users can click any Worker to access its detail page, where they can review:

- Worker functionality description
- Supported use cases
- Required input parameters
- Output data structure
- Usage examples

Users can determine whether a Worker meets their data collection requirements based on the details provided.
