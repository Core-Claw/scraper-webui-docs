## Pay per usage

You only pay for the platform resources **actually consumed** during Worker runs, with no hidden fees.

**Pay only for successful results. Failed requests are completely free.**

![Worker pricing configuration](@/assets/docs/img_4.jpg)

## How to estimate costs?

While it's difficult to predict resource consumption precisely before a run, you can always view the detailed bill after each run.

### Cost records

![Cost records](@/assets/docs/img_5.jpg)

:::note[Best practice]
We recommend running a small-scale test (e.g., scraping only a few pages). Through a trial run, you can directly evaluate resource consumption and extrapolate the budget for a large-scale run.
:::

For detailed pricing information, visit [coreclaw.com/pricing](https://www.coreclaw.com/pricing).
