Running a Worker on CoreClaw takes a few minutes and requires no code. The full step-by-step guide with screenshots lives in one place:

:::note[Full guide]
**[Quick Start — run your first Worker](/user-guide/run-worker/quick-start/)** covers the complete flow: enter the Store → find a Worker → configure inputs → Start → view results → export.
:::

## At a glance

1. **Find** a Worker in the [Worker Store](https://console.coreclaw.com/) — browse by category or search.
2. **Review** its description, input parameters, sample output, and pricing.
3. **Configure** the inputs (URLs, keywords, etc.) and run options (proxy, version, timeout).
4. **Start** the run and watch its status move `READY → RUNNING → SUCCEEDED`.
5. **Export** the results in **CSV, JSON, JSONL, XLS, XLSX, HTML, XML, or RSS**.

## Tips

:::tip[Best practices]
- Start with a small test run first to verify output and cost.
- Check the Worker's sample output before running at scale.
- Save a working run as a **Task** for reuse and scheduling.
:::
